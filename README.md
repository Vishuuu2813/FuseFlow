# WhatsFlow (FuseFlow)

WhatsFlow is a enterprise-grade, multi-tenant WhatsApp Business Automation SaaS platform. It enables businesses to automate marketing campaigns, handle customer communications, track leads in a CRM pipeline, and leverage AI for automated responses and knowledge base integration.

## Technical Stack
- **Frontend**: React, Vite, TailwindCSS v4, React Router, Redux Toolkit, TanStack Query, Framer Motion, Axios.
- **Backend**: Node.js, Express.js, Socket.io, BullMQ (Redis).
- **Database**: MongoDB (Mongoose ODM).
- **WhatsApp engine**: Baileys Multi-Device API.
- **Integrations**: Stripe, Razorpay, Cloudinary, ExcelJS, OpenAI/Gemini.

## Project Structure
- `/fuseflow`: Frontend React-Vite application.
- `/server`: Backend Node.js-Express application.
- `/artifacts`: Developer blueprints and architecture design plans.
